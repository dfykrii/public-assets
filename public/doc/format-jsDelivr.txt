🔗 FORMAT LINK jsDelivr UNTUK GAMBAR:

Repo GitHub:
https://github.com/dfykrii/public-assets/tree/main/public/images

CDN jsDelivr format:
https://cdn.jsdelivr.net/gh/dfykrii/public-assets/public/images/<NAMA-FILE>
https://cdn.jsdelivr.net/gh/dfykrii/public-assets/public/images/2_20250617_125747_0001.png

📝 Cara pakai:
1. Upload gambar ke folder: public/images/
2. Ganti <NAMA-FILE> dengan nama file lo (termasuk ekstensi)

✅ Contoh:
Nama file: `kucing-lucu.webp`
CDN jsDelivr:
https://cdn.jsdelivr.net/gh/dfykrii/public-assets/public/images/kucing-lucu.webp

Link ini bisa langsung dipakai untuk:
- <img src="..." />
- HTML, CSS, JS
- Disimpan di database
- Dan keperluan lainnya
